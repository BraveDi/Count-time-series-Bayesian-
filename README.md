This repository contains the R code used for the Real application in the paper: "Parameter estimation of Count Data models using Bayesian techniques"

## Contents
1. TLC.1.R - R codes for real analysis.
2. TLC.xlsx -  Contains the Agricultural dataset containing observations on Tomato Leaf Curl Disease (TLC) incidence.

## Requirements 
1. R version 4.5.3 or higher.
2. Packages: readxl, MASS, pscl, lmtest, AER, rstanarm, bayestestR, ggplot2, dplyr, tidyr.
